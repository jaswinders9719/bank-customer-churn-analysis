# 📊 Bank Customer Churn Analysis

## 📌 Overview

This project investigates the relationship between geographical location and customer churn in the banking sector using statistical analysis.

The dataset contains 10,000 customers across France, Germany, and Spain.

---

## 🎯 Research Question

Is there a difference in customer churn rates across different geographical regions?

---

## 🧠 Methodology

* Data cleaning and preprocessing
* Statistical analysis using Chi-Square Test
* Data visualization using stacked bar charts

---

## 📊 Key Findings

* Germany has the highest churn rate (~32%)
* France and Spain have lower churn rates (~16–17%)
* Strong statistical relationship found between geography and churn (p-value < 0.05)
